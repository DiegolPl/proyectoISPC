Resumen de la propuesta

Mi idea es tener un lugar donde registrar tu mascota para tener todo allí a mano, datos como por ejemplo: nombre, raza, vacunas puestas, fotos, etc.

Otro idea es usar este sitio/app como un lugar para encontrar al posible dueño de una mascota perdida a partir de un sistema de filtrado por lugares. En el perfil de cada mascota se habilitaría un botón para darlo como perdido que permita aparecer al principio de la lista para una mejor posibilidad de lograr ese reencuentro.

Características

  • Sistema de registro y login por dueño
  • Sección de búsqueda por zona para mascotas extraviadas (filtros)
  • Página individual por mascota
  • Dashboard por dueño para administrar la información de su/sus mascotas
  • Botón para generar alerta por mascota perdida que marque a la misma como perdida
  • Carrousel con fotos de nuevos mascotas registradas
  • Sistema de likes para mascotas
